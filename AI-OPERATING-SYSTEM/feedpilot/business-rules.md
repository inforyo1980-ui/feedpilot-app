# FeedPilot Core Rules

FeedPilot is a Shopify SEO automation SaaS.

Core Goal:
Increase recurring subscription revenue through SEO automation and automation retention.

Plans:
- Free
- Starter ($9/month)
- Growth ($19/month)

Free Plan:
- 2 optimizations every 7 days
- force upgrade after limit reached

Starter Plan:
- manual optimization enabled
- no automation

Growth Plan:
- weekly automation enabled
- automation should feel active and valuable
- recurring value must be visible in UI

Core Product Value:
- SEO issue detection
- automated optimization
- recurring improvements
- automation retention
- optimization history
- weekly insights

Never:
- fake ROI
- fake analytics
- fake traffic numbers
- inconsistent metrics
- random UI redesigns
- breaking billing logic

UI Principles:
- conversion-focused
- clean hierarchy
- strong CTA
- commercial SaaS feeling
- avoid chaotic layouts

Important:
Always preserve:
- billing flow
- Free/Starter/Growth logic
- automation cooldown logic
- Optimization History integrity
