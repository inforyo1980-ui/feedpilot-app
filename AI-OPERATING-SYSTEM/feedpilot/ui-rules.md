# FeedPilot UI Rules

Goal:
Increase paid conversion while keeping a clean SaaS dashboard.

Plan colors:
- Free: gray
- Starter: blue
- Growth: green

Hero:
- use loss framing
- show clear opportunity count
- avoid fake ROI
- CTA must match current plan state

Free UI:
- show remaining free optimizations
- after 2 optimizations per 7 days, show upgrade CTA

Starter UI:
- show manual optimization available
- show Growth upgrade CTA for automation

Growth UI:
- show automation active
- show weekly optimization status
- show Optimization History and Weekly Insight

Never:
- show fake traffic increase
- show fake revenue impact
- use inconsistent counts
- create too many CTAs
- mix Free/Starter/Growth states visually

CTA principles:
- one clear primary CTA
- upgrade CTA should be direct
- avoid vague wording
